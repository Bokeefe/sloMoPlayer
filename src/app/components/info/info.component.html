this is a slow music player made by brendan okeefe. inspired by Bohren and Der Club of Gore and the Caretaker

it creates 30 track genre playlists, plays the tracks slowly, and adds some reverb. u can further tweak the to find that syrupy sweet spot.

The reverb takes a lot of browser memory so it might take a full minute to load on a phone.

Created in Angular 7 with a Node server. I am probably looking for work so checkout my resume site >>>
